## Entry 4 — Phase B2: `09_wild_bootstrap.R` implementation (~340 lines)

Created new dedicated script for Wild cluster bootstrap. Initial implementation used **UNRESTRICTED Wild bootstrap** (y_b = y_hat + v×u from full fit), which gave nonsensical p_wild ≈ 0.5 for ALL 14 cells — because the bootstrap DGP didn't impose the null hypothesis.

**Bug fix:** Switched to **RESTRICTED Wild bootstrap** (Cameron-Gelbach-Miller 2008 / Roodman et al. 2019):
1. Fit unrestricted model → β_full, t_obs
2. Fit restricted model (drop target_coef from RHS) → β_restricted, u_restricted
3. Bootstrap: y_b = X_restricted · β_restricted + v × u_restricted (imposes H0: target_coef = 0)
4. Refit unrestricted on y_b; compute t_b
5. p_wild = mean(|t_boot| ≥ |t_obs|)

After fix, all 14 cells produce sensible p_wild values.
