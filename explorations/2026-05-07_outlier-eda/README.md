## Hypotheses to Test

1. **H1.** 4개 outcome 모두 right-skewed (꼬리 비대칭) — log/IHS 변환 정당화 근거.
2. **H2.** `imputed_payment`은 deterministic formula로 인해 두 개 mode (1.2M 정액 + area-proportional 연속) — outlier 변수 아님 재확인.
3. **H3.** 0/음수 값 비율 ≥ 5% 인 변수 존재 — 그렇다면 spec MAY → SHOULD로 격상 (Plan OQ-4).
4. **H4.** Cutoff 근처 (`abs(rv_2018) ≤ 1000`) subsample 분포가 전체와 유의하게 다르지 않음 — RDD identification 안정성.
